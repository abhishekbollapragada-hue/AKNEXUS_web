# 🚀 AK Nexus – Role-Based Management Platform

AK Nexus is a scalable and modern role-based web application designed to manage workflows, users, and organizational tasks efficiently. The platform provides dedicated dashboards for Admins, Mentors, and Employees with secure authentication and structured access control.

## 📌 Key Features

* 🔐 Secure Authentication (Firebase)
* 👥 Role-Based Access Control (Admin / Mentor / Employee)
* 📊 Interactive Dashboards for each role
* 📝 Task Assignment & Tracking System
* 📅 Attendance Management System
* 🎓 Certification & Progress Tracking
* 💬 Feedback & Communication Modules
* 🔔 Notification System (Admin → Users)
* 🛡️ Protected Routes & Guards
* ⚡ Fast Performance with Vite
* 🎨 Clean UI with Tailwind CSS

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Backend/Services:** Firebase (Authentication, Firestore, Hosting)
* **Routing:** React Router
* **State Management:** React Hooks

---

## 📂 Project Structure

```
src/
│── components/        # Reusable UI components
│── pages/             # Role-based pages (Admin, Mentor, Employee)
│── layouts/           # Dashboard layouts
│── guards/            # Route protection (role-based access)
│── hooks/             # Custom hooks
│── services/          # API & business logic
│── firebase/          # Firebase configuration
│── utils/             # Helper functions
│── App.jsx            # Root component
│── Routes.jsx         # Routing logic
```

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/abhishekbollapragada-hue/AKNEXUS_web.git
cd AKNEXUS_web
```

### 2️⃣ Install dependencies

```
npm install
```

### 3️⃣ Run the application

```
npm run dev
```

---

## 🔥 Firebase Configuration

1. Create a project in Firebase Console
2. Enable Authentication
3. Setup Firestore Database
4. Replace config inside:

```
src/firebase/firebase.js
```

---

## 🚀 Deployment

This project can be deployed using Firebase Hosting:

```
npm run build
firebase deploy
```
